# Stack-Overflow-Developer-Survey-Analysis

A Look into Recent Trends in Software Industry. This Project analyses on the recent trends in software industry by using the Stack Overflow Developer Survey Results 2019. You can download the dataset from the following link [https://www.kaggle.com/mchirico/stack-overflow-developer-survey-results-2019]. Detailed explanation of the analysis can be found in my article from the following link:[https://medium.com/@kunal_gohrani/the-future-of-software-industry-a-look-into-the-current-trends](https://medium.com/@kunal_gohrani/the-future-of-software-industry-a-look-into-the-current-trends)

## Technologies used

1.  Pandas
2.  Numpy
3.  Matplotlib
4.  Seaborn
5.  Wordcloud
